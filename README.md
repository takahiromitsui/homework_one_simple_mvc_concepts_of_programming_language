## Tech Stack
- TypeScript
- Nodejs(v16.14.0)


## Run Locally

Clone the project

```bash
  git clone https://github.com/takahiromitsui/homework_one_simple_mvc_concepts_of_programming_language.git
```

Install dependencies

```bash
  yarn install
```


Start the server

```bash
  yarn start
```
